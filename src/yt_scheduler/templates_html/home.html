New project — step 1 of 3: connect YouTube

Sign in with the Google account that owns the YouTube channel you'll post to from this project. The channel id you authenticate as becomes the project's permanent posting target.

The project row isn't created until OAuth completes successfully — you can cancel safely at any time during this step. After OAuth, the next step lets you name the project (default is your channel's display name).

New project — step 2 of 3: name

Connected to your channel. Pick a name for this project. We've defaulted to the channel's display name; you can change it now or rename later from Project Settings.

The project's URL slug is derived from this name on first save and stays fixed for the project's lifetime; renames only change the display name.

New project — step 3 of 3: social defaults (optional)

Pick which credential this project should post as on each platform. You can skip and configure these later on the project's Settings page.
